## Cyber security


# Cybersecurity Career Path Roadmap

## Technical Core Competencies
* **Systems & Networking**: Linux/Windows administration, TCP/IP stack, DNS, Active Directory architectures, and routing protocols.
* **Automation**: Scripting proficiency in Python, Bash, or PowerShell for security orchestration and log parsing.

## Specialization Tracks

### Offensive Security
* **Roles**: Penetration Tester, Vulnerability Researcher, Exploit Developer.
* **Skills**: Web/network application testing, source code auditing, Metasploit, Nmap, Burp Suite.
* **Certifications**: OSCP, PNPT.

### Defensive Security & Engineering
* **Roles**: SOC Analyst, Incident Responder, Cloud Security Architect, DevSecOps Engineer.
* **Skills**: SIEM deployment (Splunk/ELK), log analysis, Wireshark, EDR management, CI/CD pipeline securing.
* **Certifications**: GCIH, CCD, CISSP.

### Governance, Risk & Compliance (GRC)
* **Roles**: Security Auditor, Risk Assessor, Compliance Officer.
* **Skills**: Framework alignment (NIST CSF, ISO 27001, SOC 2, GDPR), enterprise risk modeling, technical policy drafting.
* **Certifications**: CISA, CRISC.

## Practical Validation
* **Platforms**: Hack The Box, TryHackMe (Advanced modules), Blue Team Labs Online.
* **Portfolio**: Git repositories containing custom security tools, automation scripts, infrastructure-as-code configurations, or detailed vulnerability write-ups.
